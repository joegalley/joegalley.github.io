Quick Summary: Extend WebSecurityConfigurerAdapter and implement configure(AuthenticationManagerBuilder auth.

In memory authentication allows you to quickly set up a simple way to authenticate users in your Spring Boot application. It creates a thin layer of security and allows you to prototype a security-aware application without setting up a full database. You can manage all aspects of authentication and authorization - including login credentials, roles, and authorities - with in-memory authentication.

To set up in-memory authentication in our app, we are going to extend the WebSecurityConfigurerAdapter class. There are two methods that we are going to override, but they both are named configure.
