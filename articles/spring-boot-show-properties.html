Spring Boot: How to List All Properties and their Values
Tags: Spring Boot Java

In Spring Boot, it may sometimes be helpful to view all of the currently loaded properties. This can be done by autowiring an instace of org.springframework.core.env.Environment into the relevant class and calling getPropertySources() on it.
