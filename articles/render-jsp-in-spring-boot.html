Rendering JSPs in Spring Boot
Tags: Spring Boot Java

Rendering JSP pages in Spring Boot applications is fairly straightforward, but there are two things you must do: include the necessary dependencies, and set up a ViewResolver.

1. Dependencies

Inside of your pom.xml file, make sure to include jstl and tomcat-embed-japser as dependencies. These are necessary to compile JSP pages. 


        <dependency>
            <groupId>javax.servlet</groupId>
            <artifactId>jstl</artifactId>
        </dependency>
        <dependency>
            <groupId>org.apache.tomcat.embed</groupId>
            <artifactId>tomcat-embed-jasper</artifactId>
            <scope>provided</scope>
        </dependency>

2. Set up a ViewResolver

    
    @Bean
    public InternalResourceViewResolver defaultViewResolver() {
        InternalResourceViewResolver resolver = new InternalResourceViewResolver();
        resolver.setPrefix("/WEB-INF/jsp/");
        resolver.setSuffix(".jsp");
        return resolver;
    }
